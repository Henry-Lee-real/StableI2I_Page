Method

StableI2I is a fidelity-oriented evaluation framework for image-to-image tasks that jointly considers the input image, output image, and editing instruction. Instead of focusing only on instruction following or perceptual quality, it explicitly evaluates whether the generated result preserves pre–post consistency in both semantic content and spatial structure. To make the evaluation adaptive across different I2I settings, StableI2I dynamically conditions its analysis on the task instruction and the visual relationship between the input and output.

Concretely, StableI2I evaluates fidelity from three complementary perspectives: Semantic Level, which measures whether the intended transformation is correctly reflected; Structure Level, which checks whether geometry and spatial layout remain stable in non-target regions; and Low-level Appearance, which focuses on texture, blur, noise, and other fine-grained perceptual changes. By combining these dimensions, StableI2I produces fine-grained and interpretable judgments for both image editing and image restoration scenarios.
